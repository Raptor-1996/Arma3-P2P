# A2M P2P Convertor

PAA to PNG converter with advanced GUI for Arma 3 modding.

## Features
- Dark carbon theme with neon glow effects
- Drag & drop support
- Batch processing
- Custom prefix/suffix
- Progress tracking
- Detailed logging
- First-time setup wizard

## Requirements
- Python 3.8+
- PyQt5
- Pillow

## Installation
1. Install Python 3.8 or higher
2. Run setup.bat
3. Place p2p_logo.png and p2p_icon.ico in the same directory

## Usage
1. First run: Configure paths to required tools
2. Drag & drop PAA files or click to browse
3. Set output options
4. Click "Convert PAA to PNG"

## Required Tools
- ImageToPAA
- Pal2PacE
- TexView
